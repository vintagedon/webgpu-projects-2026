## 5. Conventions

Staging is a workspace, not a permanent home. Move files to their target directory after review and approval. If a file stays here for an extended period, decide whether it is still active or belongs in `recycle/`.

In private repos, this directory can be tracked. In public repos, review whether staged prompts or drafts should remain private before publishing.

Files in staging can use any naming or organization that fits the work. Final structure is imposed when files move to their permanent location.
