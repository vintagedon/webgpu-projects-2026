FRONTMATTER: Always include. Type is always "one-pager." Status
reflects the idea's lifecycle: Ideation In Progress, Ideation
Complete, Charter Pending, Superseded.

A one-pager is a portable context unit: a self-contained idea
snapshot optimized for AI handoffs, human memory refresh, and
cross-model collaboration. It preserves the full reasoning arc
of an ideation session. The charter distills it into commitments;
the one-pager preserves context the charter compresses away.

VALIDATION TEST: Can a fresh AI instance with no conversation
history pick up this document and continue working? If not, the
one-pager needs more context.
-->

<!--
TEMPLATE NOTES:

The fixed elements are: header metadata, Vision, Origin, Document
Info, and Sources. Everything between Origin and Document Info is
domain-driven content with no prescribed structure.

A one-pager is written mid-conversation when an idea reaches its
saturation point. It captures reasoning that would otherwise be
lost when the session ends. The charter comes after the one-pager
and freezes commitments; the one-pager preserves the exploratory
context the charter cannot hold.

GDR outputs are inputs to one-pagers, not substitutes for them.
A GDR provides validated research data. The one-pager synthesizes
that data with session insights, architectural decisions, and
cross-project pattern recognition into a coherent idea snapshot.
